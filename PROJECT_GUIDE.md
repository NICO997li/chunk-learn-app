## 🚀 如何运行

由于当前环境没有 Node.js，你需要在有 Node.js 环境的机器上运行此项目。

### 步骤 1: 安装 Node.js

访问 https://nodejs.org/ 下载并安装 Node.js (推荐 LTS 版本)

### 步骤 2: 安装依赖

```bash
cd chunk-learn-app
npm install
```

### 步骤 3: 启动开发服务器

```bash
npm run dev
```

应用将在 http://localhost:5173 启动

### 步骤 4: 生产构建

```bash
npm run build
```

构建产物在 `dist/` 目录，可直接部署到任何静态网站托管服务。

## 🔧 可扩展性

### 轻松添加新语块
编辑 `src/data/chunks.ts`，按格式添加即可

### 自定义主题
修改 `tailwind.config.js` 的颜色配置

### 增加语块分类
扩展 `ChunkCategory` 类型和分类数据

### 接入真实语音API
替换 Web Speech API 为 Google TTS / Azure Speech

### 云端同步
接入 Firebase / Supabase 实现多设备同步

## 📝 使用建议

1. **每天坚持** - 利用间隔重复,每天复习10-15分钟
2. **诚实反馈** - 准确评估记忆程度,算法才能精准调度
3. **造句练习** - 看到语块时尝试在脑中造句
4. **大声朗读** - 使用语音功能,训练口语流利度
5. **关注场景** - 注意语块的使用场景,实际应用中回忆
