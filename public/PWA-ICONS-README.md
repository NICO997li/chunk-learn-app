# PWA Icons

需要添加以下图标文件到 `public` 目录:

- `pwa-192x192.png` - 192x192 像素图标
- `pwa-512x512.png` - 512x512 像素图标
- `apple-touch-icon.png` - 180x180 像素 Apple 图标

## 临时方案

Vercel 部署时会自动生成默认图标,或者可以使用以下在线工具生成:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## 图标设计建议

- 使用简洁的设计
- 确保在小尺寸下清晰可见
- 使用应用的主色调(蓝色 #3b82f6)
- 背景可以是纯色或渐变
